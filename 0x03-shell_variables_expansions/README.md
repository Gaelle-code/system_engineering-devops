# Shell Variables and Expansions

   This project explores environment variables, shell initialization files, and expansions in Linux.

   ## Tasks
   - 0-alias: Creates an alias for ls command
   - 1-hello_you: Prints hello to current user
   - 2-path: Adds /action to PATH
   - 3-paths: Counts directories in PATH
   - 4-global_variables: Lists environment variables
   - 5-local_variables: Lists all variables and functions
   - 6-create_local_variable: Creates local variable BEST
   - 7-create_global_variable: Creates global variable BEST
   - 8-true_knowledge: Arithmetic with TRUEKNOWLEDGE variable
   - 9-divide_and_rule: Division operation
   - 10-love_exponent_breath: Exponentiation operation
